BadgeView

Description

  • There is a notification with a red mark on the icon on the iPhone home screen. The image can be replaced with various tools. But this tool can simply draw lines.

    Border
    1px, 2px, 3px are available.

    Color settings
    Using libcolorpicker, you can easily change it to whatever color you want.

Compatibility

  • iOS 7.0 - 13.3.1
