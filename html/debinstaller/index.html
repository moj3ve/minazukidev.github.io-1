Description

This is an app that installs deb files all at once. The deb is installed by tapping one by one, but by placing the deb in
/ Applications / DebInstaller / deb, you can install the deb "ignoring dependencies" collectively. I don't think you use
it often, but it will be a useful app for those who are
backing up deb. Only compatible with unc0ver.
